# Code of Conduct

## Our Standard

This project expects respectful, technically focused collaboration.

Examples of expected behavior:

- discuss code and evidence directly
- challenge ideas without attacking people
- keep review feedback specific and actionable
- avoid posting credentials, private endpoints, or personal data

Examples of unacceptable behavior:

- harassment or personal attacks
- publishing someone else's secrets or private infrastructure details
- low-signal spam in issues or pull requests
- intentionally disruptive behavior

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributors that violate these standards.

## Reporting

Report serious conduct issues privately to the project maintainer instead of escalating in public issue threads.
